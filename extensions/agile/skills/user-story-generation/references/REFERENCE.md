# user-story-generation — reference

## Good vs bad goal lines

**Good:** As a warehouse operator, I want to scan a barcode and see stock level and location, so that I can restock quickly.

**Bad:** As a user, I want a new API endpoint for stock. (solution, not outcome.)

## Good vs bad titles

**Good:** "Scan barcode to check stock level"

**Bad:** "Add API endpoint for stock" (implementation) / "As a user I want…" (story line, not title)
